【立即开始】

1️⃣  双击运行
    SerialDebugger_ImGui.exe

2️⃣  连接串口
    - 选择串口号
    - 点击"连接串口"

3️⃣  启用可视化
    菜单栏 → "可视化" → "显示可视化界面"

4️⃣  快速测试（可选）
    使用Python脚本快速测试：
    python 测试脚本_FireWater.py
    （需要先安装：pip install pyserial）

【文件说明】

✓ SerialDebugger_ImGui.exe  - 主程序
✓ 使用说明.txt              - 详细文档
✓ 测试脚本_FireWater.py     - Python测试工具
✓ libgcc_s_seh-1.dll        - 运行时库
✓ libwinpthread-1.dll       - 线程库
✓ libstdc++-6.dll           - C++库

【核心特性】

📡 串口通信
  - 自动识别串口
  - HEX/ASCII显示
  - 定时自动发送

📊 数据可视化（NEW!）
  - 4种协议：FireWater/JustFloat/RawData/Custom
  - 5种组件：波形图/数字表盘/柱状图/仪表盘/表格
  - 16通道：独立配置、颜色、统计

🎨 拖拽布局
  - 组件库侧边栏
  - 拖拽添加组件
  - 实时配置参数

【第一次使用？】

试试这个简单流程：

1. 运行程序
2. 菜单 → 可视化 → 显示可视化界面
3. 协议配置 → 选择"FireWater"
4. 通道配置 → 勾选CH1-CH4
5. 组件库 → 点击"波形图"的"添加"按钮
6. 连接串口 → 发送数据

详细说明请阅读"使用说明.txt"
